> **Curso de IA Aplicada a Investigación Científica y Tecnológica — Nanotecnología**

Curso estructurado completo (Unidades 1–6) para aprender IA aplicada a nanotecnología — desde simulación molecular hasta sistemas multi-agente y despliegue de modelos en producción.

---

## ¿Qué es esto?

Este repositorio combina dos campos que raramente se enseñan juntos:

- **Nanotecnología Computacional**: dinámica molecular (ASE), DFT, optimización de nanopartículas, análisis XRD
- **Sistemas de IA Modernos**: LLMs, frameworks multi-agente (Google ADK 1.0, CrewAI, LangChain), protocolo A2A, RAG, despliegue en producción

El resultado es un curso único para investigadores y estudiantes en ciencias de materiales, química computacional y nanotecnología que quieren integrar IA en su trabajo.

## Inicio Rápido

```bash
git clone https://github.com/Multiagent-AI-Lab/Antigravity-Nano-Research-Multiagentic-Core.git
cd Antigravity-Nano-Research-Multiagentic-Core
setup.bat        # Windows
# o
./setup.sh       # Linux/macOS
conda activate ia_nano
jupyter lab
```

## Tecnologías Principales

- **Simulación Molecular**: ASE, RDKit, OpenMM, GPAW
- **ML/IA**: PyTorch, scikit-learn, Hugging Face
- **Multi-Agente**: Google ADK 1.0, LangChain, CrewAI, LangGraph
- **Producción**: FastAPI, Docker, OpenTelemetry, Prometheus
